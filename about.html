About Sora Hub

Sora Hub is a professional web-based arcade designed for the Web Technologies 2025 coursework. The goal was to build "something fun" that demonstrates high-level client-side programming skills.

Core Technologies

  • HTML5: Semantic structure for all game interfaces.
  • CSS3: Custom properties (variables) for the "Sora" dark theme and responsive Flexbox/Grid layouts.
  • JavaScript: Complex AI logic (Minimax for Tic-Tac-Toe), memory management, and LocalStorage for cross-game score tracking.

The Developer

Developed as a solo project to explore the intersection of UI design and algorithmic game logic.
